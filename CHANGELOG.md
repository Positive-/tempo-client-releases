# Changelog

## Tempo 0.0.7 - 2026-06-08

### New
- Replay review now includes built-in drawing tools for marking up plays directly over the replay.
- Use pointer, pen, eraser, color selection, brush size, and clear controls from the replay overlay.

### Improved
- Replay review now uses one compact overlay instead of switching between full and minimal overlay modes.
- The annotation overlay stays usable when switching between Tempo controls and the native League replay client.
- The overlay takes up less space while still keeping review tools close at hand.
